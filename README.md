# Lista de Ninjas em Java

Este projeto foi desenvolvido em **Java** com o objetivo de praticar conceitos fundamentais da **Programação Orientada a Objetos (POO)** e o uso de **estruturas de dados** da linguagem.

O programa cria uma lista de ninjas e demonstra operações como **adicionar, remover, listar e buscar elementos dentro de uma lista**.

---

## Tecnologias utilizadas

* Java
* Estrutura de dados `LinkedList`
* Programação Orientada a Objetos (POO)

---

## Conceitos aplicados

Neste projeto foram utilizados diversos conceitos importantes da linguagem Java:

* Criação de **classes e objetos**
* **Construtores**
* **Encapsulamento**
* **Getters e Setters**
* **Sobrescrita do método `toString()`**
* Uso da coleção **LinkedList**
* **Laço de repetição (for-each)**
* Manipulação de objetos dentro de listas

---

## Estrutura do projeto

ListaNinja.java
Classe principal responsável por:

* Criar a lista de ninjas
* Adicionar ninjas
* Listar ninjas
* Remover ninjas
* Buscar ninja por índice

Ninjas.java
Classe que representa um ninja contendo os seguintes atributos:

* nome
* idade
* aldeia
