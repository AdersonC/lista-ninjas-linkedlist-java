# Lista de Ninjas com LinkedList em Java

Este projeto foi desenvolvido em Java para demonstrar o uso da estrutura de dados LinkedList para armazenar e manipular objetos.

O programa cria uma lista de ninjas e realiza diversas operações, como:

- Adicionar ninjas à lista
- Listar ninjas
- Inserir um ninja no início da lista
- Remover ninjas da lista
- Acessar um ninja por índice

## Tecnologias utilizadas

- Java
- IntelliJ IDEA

## Conceitos aplicados

- Programação Orientada a Objetos
- Criação de classes e objetos
- Estrutura de dados LinkedList
- Manipulação de coleções
- Laço de repetição for-each

## Como executar o projeto

1. Baixe ou clone o repositório
2. Compile os arquivos Java
3. Execute o programa

Exemplo:

javac ListaNinja.java  
java ListaNinja
